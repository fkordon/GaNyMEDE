For FUN (open eds), there should be the following files:

- syllabus.html: that describes the syllabus. e propose a typical content structure. Please do not remove the "section"
  tag that is used in the CSS apparently. File is MANDATORY.

- configure-mooc.sh: that contains the global information to be used in the MOOC. This file is in fact a bash script
  for implementation convenience. File is MANDATORY.

- doc-pedagogiques.html: that contains a short description of some documents that can be accessed by the students for
  practice or additional information (file is optional)

If your syllabus refer to images, please use png or jpg files.
